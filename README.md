# Dicee

Приложение-кубик для игры в кости. При нажатии на кнопку выпадают случайные значения от 1 до 6.

## Функционал

- Два игральных кубика
- Случайный результат при каждом броске

## Технологии

- Swift
- UIKit
- UIImageView
- randomElement()
